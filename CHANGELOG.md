# Changelog

All notable changes to Argenté will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-29

### Added
- Initial release of Argenté e-commerce platform
- Complete product catalog with 12+ products
- Shopping cart with localStorage persistence
- Wishlist functionality
- Product filtering by category and price range
- Product sorting (newest, price, name)
- Multi-step checkout process (Cart → Shipping → Payment → Confirmation)
- Contact form with validation
- Newsletter subscription
- Toast notifications for user feedback
- Search functionality across products
- Recently viewed products tracking
- Responsive design for all screen sizes
- SEO optimization with meta tags
- Premium brown/beige color scheme
- Social media links with brand colors
- Trust badges (secure payment, 24/7 support)
- Multiple payment options (Card, UPI, Net Banking, COD)
- Scroll to top button
- Product ratings and reviews display
- Size selection for products
- Discount badges on products
- Glassmorphism effects throughout UI

### Technical Features
- React 18.2.0 with functional components and hooks
- React Router DOM for navigation
- Context API for state management
- Vite 5.0.8 for fast development and building
- Custom CSS with variables for theming
- Code splitting and lazy loading
- Performance optimized build configuration
- ESLint configuration
- EditorConfig for consistent code style
- Comprehensive documentation

### Documentation
- README with setup instructions
- CONTRIBUTING guidelines
- Environment variables example
- MIT License
- Detailed project structure
- Code style guidelines
